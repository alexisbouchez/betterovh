## UI Components

### Tables

| Component | Description |
|-----------|-------------|
| `InstancesTable` | Display and manage cloud instances |
| `VolumesTable` | Display and manage storage volumes |
| `NetworksTable` | Display private networks |
| `SSHKeysTable` | Display and manage SSH keys |

### Forms

| Component | Description |
|-----------|-------------|
| `CreateVolumeForm` | Create new storage volumes |
| `CreateSSHKeyForm` | Add new SSH keys |
| `CreateNetworkForm` | Configure private networks |

### Feedback

| Component | Description |
|-----------|-------------|
| `ErrorState` | Reusable error display with retry |
| `EmptyState` | Reusable empty state display |
| `Spinner` | Loading spinner indicator |
| `NotificationCenter` | Toast notifications |

### Dashboard

| Component | Description |
|-----------|-------------|
| `StatsCards` | Dashboard statistics overview |
| `ActivityFeed` | Recent activity display |
| `ResourceChart` | Resource usage visualization |

## Base UI Components

BetterOVH uses [Base UI](https://base-ui.com/) for unstyled primitives:

- `Button` - Button with variants (default, outline, ghost, destructive)
- `Input` - Text input with validation states
- `Select` - Dropdown selection
- `Dialog` - Modal dialogs
- `Tabs` - Tab navigation
- `Table` - Data tables

## Component Patterns

### Loading State

```tsx
function MyComponent() {
  const { data, isLoading, error } = useMyQuery()

  if (isLoading) {
    return <Skeleton />
  }

  if (error) {
    return <ErrorState error={error} onRetry={refetch} />
  }

  if (!data?.length) {
    return <EmptyState title="No items" />
  }

  return <MyTable data={data} />
}
```

### Form with Validation

```tsx
function MyForm({ onSubmit }) {
  const [value, setValue] = useState('')
  const [touched, setTouched] = useState(false)

  const error = touched && !value ? 'Required' : undefined

  return (
    <form onSubmit={handleSubmit}>
      <Input
        value={value}
        onChange={(e) => setValue(e.target.value)}
        onBlur={() => setTouched(true)}
        aria-invalid={!!error}
      />
      {error && <p className="text-destructive">{error}</p>}
    </form>
  )
}
```
